## SQL Concepts Demonstrated

---

## Datasets

The queries operate on structured relational datasets including:

- students
- grades
- faculty
- products
- categories

These tables represent common relational patterns such as
one-to-many relationships, foreign keys, and time-based records.

Synthetic datasets are used to demonstrate logic without relying
on proprietary or sensitive data.
